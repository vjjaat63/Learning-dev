Understanding HTML Hyperlinks

What is a Hyperlink?

A hyperlink, or simply a "link," is a clickable element on a web page that directs users to another location. This could be another web page, a section within the same page, or even an external resource. Hyperlinks are created using the <a> (anchor) tag in HTML.

Creating Internal Links

Internal links direct users to another page or section within the same website. They help users navigate through different parts of your site.

Example of Linking to Another Page

To link to another page within the same website, you use a relative URL in the href attribute:

<a href="about.html">About Us</a>

In this example, clicking "About Us" will take the user to about.html on the same domain.

Example of Linking to a Section Within the Same Page

To link to a specific section of a page, you first need to assign an id to the target element:

<h2 id="section1">Section 1</h2>

Then, create a link that points to this section:

<a href="#section1">Go to Section 1</a>

Clicking this link will scroll the page to the element with id="section1".

Creating External Links

External links direct users to a page or resource outside of your website. These links usually lead to other websites or external resources.

Example of Linking to an External Website

To link to an external website, you use the full URL in the href attribute:

<a href="https://www.example.com">Visit Example.com</a>

Clicking "Visit Example.com" will open the specified URL in the same tab.

Example of Opening Links in a New Tab

To make an external link open in a new tab, use the target="_blank" attribute:

<a href="https://www.example.com" target="_blank">Open Example.com in a New Tab</a>

Including rel="noopener noreferrer" is recommended for security reasons:

<a href="https://www.example.com" target="_blank" rel="noopener noreferrer">Open Example.com in a New Tab</a>
