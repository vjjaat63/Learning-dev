About website

Welcome!This page is dedicated to the official events management of the nit Warangal.This is the official page for all the event managers.Every year an event festival is held including various events in which students compete with one another to achieve the victory.

Purpose

The main purpose of this page is to connect with all who are willing to participate in the events holding by the college within the year. This page provides information and updates about the upcoming events and all the necessary details about it.

Functions

The main functions of the website are allowing the student to get to know about all events and further assists in the following ways.
  • Registration for an event
  • Ask query about the events
  • Code of conducts of the events
  • Two way interaction with the event manager
